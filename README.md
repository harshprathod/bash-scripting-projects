## ğŸ Getting Started

### Prerequisites

- Linux/Unix system (CentOS, Ubuntu, etc.)
- Bash 4.0 or higher
- Git installed
- Basic terminal knowledge

### Installation

```bash
# Clone the repository
git clone git@github.com:harshprathod/bash-scripting-projects.git

# Navigate to the directory
cd bash-scripting-projects

# Check available projects
ls -la beginner/
```

### Running a Script

```bash
# Navigate to any project
cd beginner/01-system-info-dashboard

# Read the documentation
cat README.md

# Make script executable
chmod +x system_info.sh

# Run the script
./system_info.sh
```
