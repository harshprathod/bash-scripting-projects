# Shell Scripting for DevOps

This repository contains basic to intermediate Bash shell scripts
focused on Linux automation and DevOps-oriented tasks.

## What this repo covers
- Shell scripting fundamentals (variables, conditions, loops)
- File and log management
- Disk and system monitoring
- Automation scripts (cleanup, backup, health checks)
- CSV and text processing
- Menu-driven and utility scripts

## Purpose
This repository is created for:
- Practicing real-world shell scripting
- Preparing for Linux / DevOps interviews
- Building automation mindset using Bash

## Tech
- Bash
- Linux utilities (grep, awk, sed, find, df, du, etc.)

## Note
Scripts are written with clarity and safety in mind, avoiding
hardcoded values and unsafe operations.
